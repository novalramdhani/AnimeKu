# Ani Cast
Ani Cast - Anime List & Trending App. (Powered by [Jikan API](https://docs.api.jikan.moe/))

see [Movie Cast](https://movie-cast.herokuapp.com) alternative from Ani Cast about movie website.

## Demo

## Features
- List anime (with)
    - trending anime
    - upcoming anime
    - popular anime
    - recomendation anime
- Detail anime
- List characters
- List seiyuu (people voice)
- Search anime

## Tech Stack
- Laravel
- Tailwind CSS
- Livewire
- Alpine.js
